Pure Javascript Geotagging tool

The geotagged images can be dragged to the map. User can drag several images simultaneously. They will appear on the map with red pin and cannot be moved.

The images without geotag can be positioned on the map using shift keyboard key and moving the mouse (whithout pressing the mouse key) over the image. After positioning the image can be saved from the info window.

For both tagged and not tagged images you can click on the image to read all the metadata (EXIF tags) written in the image file.

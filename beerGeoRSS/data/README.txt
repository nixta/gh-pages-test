//Munich Beer Gardens
-This application displays XML-(GeoRSS) data which is collected from "http://www.biergärtenmünchen.de"
-All graphics are filtered on price per "Maß" beer and set on a separate graphics layer
-The extent is set automatically on all graphis which are on the layers
-The "home-Button" is zooming to the auto-extent
-The geocoder places a flag-Icon based on the "search.results" extent and removes the graphic onPan
